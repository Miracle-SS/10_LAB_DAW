# Laboratorio 10 - Angular

## Juego del Ahorcado

Proyecto desarrollado para el lab - curso de Desarrollo de Aplicaciones Web utilizando Angular.

### Descripción

La aplicación implementa un juego del ahorcado que permite:

* Seleccionar una palabra aleatoria.
* Mostrar letras ocultas.
* Ingresar letras por teclado.
* Validar aciertos y errores.
* Mostrar mensajes de victoria o derrota.
* Reiniciar el juego.
* Mostrar letras utilizadas.

### Tecnologías utilizadas

* Angular
* TypeScript
* HTML
* CSS

### Ejecución del proyecto

Instalar dependencias:

```bash
npm install
```

Ejecutar la aplicación:

```bash
ng serve
```

Abrir en el navegador:

```text
http://localhost:4200
```
